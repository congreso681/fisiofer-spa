# Fisiofer SPA - Sistema de Gestión Clínica

Sistema web para la gestión de citas, historias clínicas y administración de un centro de fisioterapia.

## 🚀 Tecnologías

- HTML5
- CSS3 (con sistema de diseño propio)
- JavaScript Vanilla
- API REST (backend desplegado en Railway)
